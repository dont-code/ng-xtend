Plugin Tester

This project allows you to load and test in live your developed components for the ng-xtend framework

Test your components by 1. Building the xt-components framework

    npm install -g @microsoft/rush
    rush update
    rush build
  1. Starting your plugin, for example, the web-plugin
  cd ../plugins/xt-web
  ng serve web-tester
  1. Then running the xt-plugin-tester with ng serve plugin-tester
  1. Typing the plugin url in the url bar:
  1. Then pressing load button will execute the plugin
  1. Go to the test page, and selects your component to test
  1. the component is then run in editing and viewing mode, allowing you to test!
