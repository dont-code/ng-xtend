ng-xtend Framework

Build extensible Angular applications with dynamic plugins. Add features at runtime, keep your core small, and let teams extend your UI safely.

Quick start Features Demos Docs

Why ng-xtend?

Most successful platforms succeed because they are easy to extend. ng-xtend brings that same idea to Angular: define a clear data model, expose render points, and let plugins provide the right UI and behavior.

It is especially useful when you want to:

Reduce boilerplateReuse generic components instead of hand-coding every field and screen.
Load features dynamicallyInstall plugins at runtime without rebuilding the host application.
Scale teams safelyLet different teams ship extensions without tightly coupling their code.

Key Features

  • Define once, render anywhere: focus on your data structures; let plugins handle the UI.
  • Dynamic loading: load and install plugins at runtime — no rebuild required.
  • Storage agnostic: save, load, and update data to any backend or storage.
  • Seamless interop: plugins can work together even when they do not know each other.
