# XtIdbStore

This plugin is part of the [ng-xtend framework](https://github.com/dont-code/ng-xtend/blob/main/README.md).

It enable IndexedDB usage as a backend for [xt-store](https://github.com/dont-code/ng-xtend/xt-store).
